# Glosario de Términos

> Dominio: diagnóstico por imágenes y gestión de turnos.

| Término | Definición |
|---|---|
| **Callbell** | CRM de mensajería multiagente usado por el instituto para gestionar WhatsApp |
| **Card** | Representación visual de un caso en el panel web |
| **Claude API** | API de Anthropic para el modelo de IA que analiza conversaciones |
| **Closing reason** | Razón por la que se cierra un caso (12 valores definidos) |
| **Copago** | Porcentaje que paga el paciente sobre el costo del estudio (según obra social) |
| **CT / TAC** | Tomografía Axial Computarizada |
| **CBCT** | Cone Beam CT — tomografía dental de haz cónico |
| **Flag** | Indicador automático sobre un caso (ayuno, copago, orden incompleta, etc.) |
| **IOMA** | Instituto de Obra Social de la Provincia de Buenos Aires |
| **Medicina Nuclear** | Especialidad en Chiclana 385: PET CT, SPECT CT, centellogramas, etc. |
| **RIS** | Radiology Information System — sistema de información radiológica (IT SOS) |
| **RLS** | Row Level Security — políticas de seguridad a nivel de fila en Supabase |
| **Sede** | Ubicación del turno: Lavalle 11 (general) o Chiclana (Medicina Nuclear) |
| **Tipo de caso** | Clasificación A–K (11 tipos según el flujo de resolución) |
| **Token IOMA** | Número generado por la app móvil de IOMA necesario para realizar la práctica |
